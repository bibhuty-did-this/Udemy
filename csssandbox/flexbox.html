Flexbox:

  • Modern layout in CSS3
  • "flex" is a value for the `display` property
  • Replaces float & is much more elegant to work with
  • Aligns items horizontally and vertically
  • Flex items can be reordered via CSS
  • display: flex;/* Creates flex container */
  • All direct child items are child items
  • By default it'll align the items horizontally

      • Different alignment properties are

    • justify-content: Align across main axis(horizontal)
    • align-item: Align items along cross axis(vertically)
    • align-content: Align when extra space in cross axis
